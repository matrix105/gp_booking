## Running the different ends

### Start virtual environment

From main directory

`source env/bin/activate`

### Run Backend

From main directory

`cd gp_booking`

`python3 manage.py migrate`

`python3 manage.py runserver`

### Run Frontend

From main directory

`cd frontend`

`npm i` or `yarn`

`npm start` or `yarn start`
